# 🐱 Catify

Catify is a Chrome Extension built with **Manifest V3** that instantly replaces every image on a webpage with adorable cat images.

---

## ✨ Features

- 🐱 Replace webpage images with cute cats
- 🔄 Restore original images
- ⚡ Instant image replacement
- 🎨 Modern popup interface
- 📦 Uses bundled local cat images
- ✅ Manifest V3 compliant

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- Chrome Extensions API
- Manifest V3

---

## 📂 Installation

1. Clone this repository.
2. Open `chrome://extensions/`
3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the Catify project folder.

---

## 🚀 Usage

1. Open any webpage containing images.
2. Click the **Catify** extension.
3. Press **Catify Page**.
4. Enjoy adorable cats everywhere!
5. Click **Restore Images** to restore the original webpage.
